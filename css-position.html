CSS Position

stays put - Static positioning leaves the element where it is in the document flow and ignores any offset.

placed relative to initial location - Relative positioning leaves the element where it is in the document flow, but allows you to move the element with offset properties.

placed relative to parent - Absolute position takes the element out of the document flow. It still stays where you left it if there is no offset specified. If there is an offset it moves to that exact position relative to the top corner of the page. To make it more complicated, though, if one of its ancestors has position relative then it will be positioned relative to that element's top corner not the corner of the whole page. The blue section illustrates this case.

placed relative to window (doesn't scroll) - Fixed position takes the element out of the document flow. It still stays where you left it if there is no offset specified. If there is an offset it moves to that exact position relative to the top corner of the window. The element stays in the same spot on your screen even if you scroll the page.
